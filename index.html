O que são tags HTML?

Em HTML a tag é usada para criar um elemento. O nome de um elemento HTML é o nome usado entre colchetes angulares como "< p >" para criar parágrafos.

Ex:<p>Este é um parágrafo.</p>
Note que ao fechar uma tag ela tem uma barra antes de seu nome, " < / p > ", e que em elementos vazios a tag final não é requirida e nem permitida. Se os atributos não forem mencionados, os valores padrões serão usados em cada caso.

O que é um Browser?

Um browser, ou navegador, é um programa de computador que permite que você acesse e visualize informações na internet, como textos, imagens, vídeos e áudio. Ele funciona como uma "ponte" entre você e a World Wide Web. Para entender como ele funciona, imagine que a internet é uma biblioteca gigantesca, e cada site é um livro. O browser é o bibliotecário que busca o livro que você pede (por exemplo, quando você digita um endereço como www.google.com) e o traz para que você possa lê-lo na tela do seu computador ou celular.

Servidor

O que é um Servidor?

Voltar

Um servidor é um computador ou um sistema de software que fornece dados, recursos ou serviços para outros computadores, chamados de clientes, através de uma rede. A palavra "servidor" vem do verbo "servir", e é exatamente isso que ele faz: ele serve as requisições dos clientes. Para simplificar, imagine um restaurante. O servidor é como o garçom, que recebe o pedido de um cliente (você) na mesa, vai até a cozinha (o servidor) buscar o prato (os dados do site, o arquivo, etc.) e o entrega de volta na sua mesa (o seu computador).
